Este é o README para o trabalho de Estrutura de Dados II, que pode ser executado pelo console da IDE. O programa faz uso das seguintes bibliotecas, que precisam ser instaladas via pip:

art: Para gerar arte ASCII no console.
Digraph: Para criar e manipular grafos direcionados.
Pillow: Para trabalhar com imagens e processamento de imagens.

Certifique-se de ter o Python instalado no seu sistema antes de prosseguir.

Instalação
Siga as instruções abaixo para instalar as bibliotecas necessárias usando o gerenciador de pacotes pip.

Abra o terminal ou prompt de comando.
Execute o seguinte comando para instalar a biblioteca art:

pip install art

Executando o Programa
Você pode executar o programa através do console da sua IDE. Certifique-se de estar no diretório correto onde os arquivos do projeto estão localizados.

Abra o console da sua IDE.

Navegue até o diretório onde o arquivo Main.py está localizado usando o comando cd.
Execute o seguinte comando para iniciar o programa:
python Main.py
